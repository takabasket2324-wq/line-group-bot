# line-group-bot

LINEのグループチャットで、AIが自動で質問に答えてくれるBotです。
プログラミングの知識は不要です。AIがセットアップを案内します。

---

## どんなことができる？

- LINEグループに Bot を招待すると、ユーザーの質問に自動で回答します
- 管理者（あなた）は同じグループにいるので、いつでも会話を見守れます
- Bot の性格や回答ルールは、スプレッドシートで自由に変更できます
- よくある質問と回答をスプレッドシートに登録すると、Bot の回答精度が上がります

---

## 始める前に必要なもの

**Node.js** — https://nodejs.org から「LTS」版をダウンロードしてインストールしてください。

### Mac の場合
1. ダウンロードした `.pkg` ファイルをダブルクリック
2. 「続ける」→「続ける」→「同意する」→「インストール」と進む
3. パスワードを求められたら、Macのログインパスワードを入力
4. 「インストールが完了しました」と表示されたら完了

### Windows の場合
1. ダウンロードした `.msi` ファイルをダブルクリック
2. 「Next」→「I accept the terms...」にチェック →「Next」→「Next」→「Next」→「Install」と進む
3. 「ユーザーアカウント制御」のダイアログが出たら「はい」をクリック
4. 「Finish」をクリックして完了

> Node.js のインストールがわからない場合は、AIチャットで聞いてください。

---

## 始め方

### 1. このフォルダを Claude Code で開く

Claude Code デスクトップアプリで、このフォルダ（`line-group-bot`）を開いてください。

### 2. 「セットアップして」と送る

チャット欄に以下を入力して送信:

```
セットアップして
```

AIが1ステップずつ案内してくれます。あとは案内に従うだけです。

---

## セットアップで必要になるもの

AIが案内しますが、以下のアカウントが必要です:

- **LINE Developers アカウント** — https://developers.line.biz/
- **Google Cloud アカウント** — https://console.cloud.google.com/
- **Anthropic アカウント（Claude API）** — https://console.anthropic.com/
- **Render.com アカウント** — https://render.com/

全て無料で始められます。

---

## セットアップ後の使い方

1. LINEアプリで「あなた + ユーザー + Bot」の3人グループを作る
2. ユーザーがメッセージを送ると、Bot が自動で回答する
3. あなたはグループ内の会話を見守り、必要なときだけ介入する

---

## Bot のカスタマイズ

スプレッドシート「line-group-bot 管理」で Bot の動作を変更できます:

- **「システムプロンプト」シート**: Bot の性格・回答ルールを変更
- **「ナレッジ」シート**: よくある質問と回答を登録して回答精度を向上

---

## よくある質問

**Q: Bot の返答内容を変えたいです**
A: スプレッドシートの「システムプロンプト」シートを編集してください。5分以内に反映されます。

**Q: Bot が反応しなくなりました**
A: Render.com のダッシュボードで「Logs」を確認してください。エラーがあれば Claude Code のチャットに貼り付けてください。

**Q: 「node が見つかりません」というエラーが出ます**
A: Node.js のインストール後に Claude Code を再起動してください。
